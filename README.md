* 안구건조증이 미치는 요인 분석 및 예측모델 생성([보기](https://www.miricanvas.com/v/14miic2))
> kaggle의 안구건조증 데이터셋을 이용해 변수의 분포를 시각화하고 가설을 세운 후 성향점수를 분석함. 안구건조증 여부와 연관이 있는 변수를 검정하고 교호작용 추가여부를 검정히며 로지스틱 회귀모형을 적합함(전진선택법과 우도비 검정 사용)
* 고독사 예측 모델 구축을 위한 데이터마이닝 분석([보기](./고독사_데이터마이닝.pdf))  
> 여성가족부 대회에서 쓰였던 데이터셋과 KNN모델을 데이터마이닝기법을 배운 후 더 발전시켜 분석함. 더 나아가 훈련셋 데이터와 테스트데이터를 7:3으로 나눈 후 KNN, 나이브베이즈, 분류나무을 반복 적합하고 양성예측도, 민감도, 특이도, F1-score, AUC를 이용해 어떤 모델이 더 나은지 판단기준을 제시함
* DKU 시계열데이터 경진대회에 참여해 산업현장에서 수집된 고해상도 시계열 센서데이터를 기반으로 공장전체의 시간 단위 전력 소비량을 저일하게 예측하는 AI모델을 개발([보기](./2025DKU경진대회_최종결과보고서.pdf)) 
> 변수파악, EDA, 전처리, 시각화등을 한 후 시간 리샘플링과 주요 파생피처를 생성해 이상치를 탐지함. 그 후 LightGBM, SARIMAX, XGboost, CatBoost, ARIMA, Transfomer, LSTM, Bi-LSTM, Delta-LSTM에 대해 알아보고 단독이나 앙상블 사용해 예측 모델을 설계함
